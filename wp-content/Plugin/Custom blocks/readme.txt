== Description ==
This plugin is made to add custom blocks to the Gutenberg editor you would typically wouldnt find in the WordPress website editor.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start using the new blocks in the editor

== Frequently Asked Questions ==
= Can I add my own blocks? =
Yes! You can extend the plugin by registering additional blocks in the `blocks/` directory.
